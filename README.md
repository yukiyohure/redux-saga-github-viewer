# redux-saga-github-viewer

## どんなアプリ？
Githubリポジトリにあるissueなどを一覧化し、編集もできるアプリ。

## なんで作ったの？
主に以下技術の学習のため

- React,js:JSで作られた、コアの機能として差分検出をトリガーにUIの更新を行うライブラリ
- Redux:状態管理のフレームワーク
- Redux-Saga:Reduxを用いたプロジェクトで副作用のある状態の更新を可能にするライブラリ
